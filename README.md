# My Word

Cement your promises in blockchain.
If you fail, you lose 20%, otherwise you get it back.


## Tech

* Web3 v1.0 [[Docs](https://web3js.readthedocs.io/en/1.0/index.html)]
* Solidity (solc)


## Development

```sh
npm install -g ganache-cli
ganache-cli
# run the script
npm run start
```
